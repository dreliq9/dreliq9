## Production MCP servers

- **[fcp-mcp](https://github.com/dreliq9/fcp-mcp)** — Final Cut Pro MCP. 88 tools across FCPXML editing, live FCP control via AppleScript, ffprobe-backed media analysis, and parametric character puppets.
- **[caid-mcp](https://github.com/dreliq9/caid-mcp)** + **[CAiD](https://github.com/dreliq9/CAiD)** — Validated 3D CAD modeling for AI agents. `ForgeResult` validation catches silent OCCT failures with volume tracking. Backed directly by OpenCASCADE — no CadQuery dependency.
- **[archi](https://github.com/dreliq9/archi)** — AI-native architectural design with live IRC-2021 code compliance. TreemapSolver auto-layout, photorealistic interior rendering, multi-format export (SVG, DXF, glTF, IFC).
- **[adclip](https://github.com/dreliq9/adclip)** — Advertising creative generation (copy, images, video) from structured briefs.
- **[declip](https://github.com/dreliq9/declip)** — AI-native video editing toolkit with MCP server.
- **[youtube-mcp-v2](https://github.com/dreliq9/youtube-mcp-v2)** — Disciplined YouTube MCP. Skeleton-first research, validated transcripts, frame extraction, versioned cache.

## Agent infrastructure

- **[llm-intentions](https://github.com/dreliq9/llm-intentions)** — An open protocol for turning Android apps into MCP tool servers. CapApp model + Hub gateway + experimental Intent Mesh multi-LLM routing. Running in production on a real device.
- **[empanel](https://github.com/dreliq9/empanel)** — Anti-sycophancy multi-evaluator engine. Lens diversity, isolated weighing, finding fingerprint synthesis. Designed to surface real disagreement instead of laundered consensus.
- **[qa-hard](https://github.com/dreliq9/qa-hard)** — Claude Code plugin built on empanel. Rigorous multi-evaluator code review with no API key required.

## In progress

- **specialist-loop** — Dual-loop automated SLM specializer. Loop 1 generates curriculum and training conversations; Loop 2 fine-tunes a small model that compounds domain expertise over time.
- **kipilot** — KiCad PCB design agent with crowd-sourced design corpus and reputation-tracked auditing. The runtime agent will be the small model trained by specialist-loop — defensible product, not a frontier-model wrapper.
